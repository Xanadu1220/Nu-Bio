# 

BootCamp 雙系統安裝

### Step .1 下載 Windows 最新的映像檔 [網址](https://www.microsoft.com/zh-tw/software-download/windows10ISO)
> 選取版本->Windows 10[選最新版本]->確認->選擇語言->繁體中文->確認->64-bit下載
### Step .2 打開 "啟動輔助程式"(Bootcamp) 並選擇已下載的Windows 映像檔
> 使用啟動輔助程式時不可接任何外接硬碟 EX.隨身碟、行動硬碟
通常會自動選取，不必再次選取。
### Step .3 調配安裝空間 ＊所調整的安裝空間大概需減掉32GB 因為安裝完系統大小約為32 GB
> 依照自己的需求調配
### Step .4 放置數分鐘後就會進入Windows安裝頁面

## Done !
# (3) 重置MacOS
### Step .1 自行備份重要檔案
> 常見的備份位置
> * 桌面
> * 文件
> * 使用者
> * 下載
### Step .2 重新開機並在一開機時按著 Command + R
> 其他按鍵組合
> * Command（⌘）-Option-R：透過 Internet 從「macOS 復原」啟動。
> * Shift（⇧）鍵：以安全模式啟動。會在使用韌體密碼時停用。
> * Option（⌥）鍵或 Alt 鍵：啟動至「開機管理程式」，這裡可以選擇其他可用的啟動磁碟或卷宗。如果 Mac 使用韌體密碼，系統會提示您輸入密碼。

### Step .3 進入 "磁碟工具程式" 選擇內部硬碟的 APPLE SSD
> 清除並選擇AFPS格式，並依照需求自訂名稱。
### Step .4 點擊完成
### Step .5 回到 Mac工具程式點選 "重新安裝MacOS"
> 這個步驟會依照電腦硬體、網路速度，影響安裝速度 1~2個小時不等。
## Done !

# (4) 善用Spotlight
### Step .1 進到系統偏好設定->鍵盤->快捷鍵->輸入方式->選擇上一個輸入方式 調整為 Command + Space
### Step .2 進到系統偏好設定->鍵盤->快捷鍵->Spotlight->顯示Spotlight搜尋 調整為option + Space
> 平時想要快速開啟某程式時，可以使用此方式快速開啟程式、文件檔案，或是臨時需要查什麼資料可以直接輸入並且選擇以Safari搜尋！

# (5) 軟體當機沒有反應關閉全部軟體的方法
### Step .1 Option + Command + Esc 開啟強制關閉應用程式
### Step .2 全部關閉連同Finder
> 此功能即是 "工作管理員" 透過此方法可以快速將沒有反應的軟體強制關閉。
